# Text Mining

Isto pode ser feito de diferentes formas, desde o uso de expressões regulares até o uso de machine learning e deep learning.

Rotinas:
* REGEX
* Sentiment Analysis
* Word Clouds
* Deploy de modelo de ML para classificação de texto
